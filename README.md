# Coin-Flow – Personal Finance Management System (MERN Stack)

## Project Description
Coin-Flow is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application helps users track, manage, and analyze their income and expenses efficiently. It provides a simple and intuitive interface for recording financial transactions and visualizing spending patterns.

This project is developed as an academic Full Stack Development (FSD) project.

## Objectives
- To simplify personal expense and income tracking
- To provide clear financial insights using analytics
- To allow users to manage transactions securely
- To demonstrate real-world MERN stack implementation

## Key Features
- Secure user authentication and authorization
- Add, update, and delete transactions
- Categorization of income and expenses
- Date-based and type-based filtering
- Analytics using charts and tables
- Responsive UI for desktop and mobile devices

## Technology Stack

### Frontend
- React.js
- React Router
- Bootstrap & CSS
- Axios

### Backend
- Node.js
- Express.js
- JSON Web Tokens (JWT)

### Database
- MongoDB
- Mongoose ORM

## Technical Architecture
Coin-Flow follows a client-server architecture:
- React handles the presentation layer
- Express and Node.js manage API requests
- MongoDB stores user and transaction data
- JWT ensures secure authentication

## How to Run the Project Locally

### Frontend
1. Navigate to the frontend directory
2. Install dependencies:
   `npm install`
3. Start the frontend:
   `npm start`
4. Open http://localhost:3000 in the browser

### Backend
1. Navigate to the backend directory
2. Install dependencies:
   `npm install`
3. Start the backend:
   `npm run dev`
4. Server runs on http://localhost:5000

## Environment Variables
Create a `.env` file in the backend folder and configure:
- `MONGO_URL` – MongoDB connection string
- `PORT` – Server port number
- `JWT_SECRET` – Secret key for authentication
